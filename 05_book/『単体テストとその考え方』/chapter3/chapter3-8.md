# なぜ統合テストを行うのか？

## 統合テストとは？
### 単体テスト以外のテスト
- 単体テストの性質
  - 1単位のふるまいを検証すること
  - 実行時間が短いこと
  - 他のテストケースから隔離されて実行されること
## テストピラミッド
### 単体テストの役割
検証する内容のほとんどは単体テストに持たせる
- 可能な限りの異常ケースを検証する
### 統合テストの役割
- 1, 2件の包括的なhappy path
- 単体テストでは実施できない異常ケース
